# Single-page-cv

A single page CV created using HTML only.

## Info about the HTML file

1. Semantic tags used
2. SEO meta tags used
3. Open Graph tags used
4. Favicon also included

## How to run the project

1. Clone the given repository on your local device
2. Click on the 'index.html' file and open it in your default browser
